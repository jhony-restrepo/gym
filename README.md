# Gym Management System 🏋️‍♂️

Este es un proyecto de software para la gestión de un gimnasio, desarrollado como parte de mi formación en ingeniería. Incluye un sistema de base de datos para controlar usuarios, planes y asistencias.

## 🚀 Tecnologías Utilizadas

* **Frontend:** React con Vite
* **Backend:** Node.js
* **Base de Datos:** SQL Server
* **Estilos:** CSS / Tailwind (según lo que uses)
